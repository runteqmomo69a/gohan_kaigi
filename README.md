# gohan_kaigi
卒業制作：お店選び共有アプリ

## サービス概要
複数人で「どこのお店に行くか」を簡単に決められる、提案・投票型のごはん/カフェ選び共有アプリ。  
URL共有でみんなが気になるお店を登録し、いいねで比較することでみんなの「行きたい！」を可視化し最適なお店を選べる。  
幹事の負担を減らしみんなでスムーズに楽しくお店を決める体験を提供する。  

## このサービスへの思い・作りたい理由
前職で複数人のランチや飲み会のお店を決めるとき、幹事役として「全員の意見を集めるのが大変」「決めるまでが時間かかる」というストレスがあった。
また友達とのランチやカフェ選びでも、「どこがいい？」「行きたいお店ある？」というやり取りが長引いたり、
せっかく候補を出してもスクショやURLがトークに埋もれてしまって見返しにくかったり、お店の情報を一目で比較しづらいことで不便さを感じていた。
そんな経験からみんなの意見を簡単に集めて比較できる “ごはん会議アプリ” があれば絶対便利という思いでこのサービスを作ることにした。

## ユーザー層について
- 職場のランチや飲み会でお店を決める人
- 友達・恋人・家族とごはんやカフェに行く人
- カフェ巡りやランチが好きな人

## サービスの利用イメージ
- ユーザーはログインしてイベント（例：金曜ランチ会）を作成。  
- イベントを作ると、参加者用の ユニークURL が発行される。  
- LINEやInstagram DMでURLを共有すると、誰でもそのページへ参加可能。  
- 参加者は「気になるお店を登録」から店名・URL・メモ等を追加。  
- お店候補にはいいねとコメントがつけられる。  
- いいね数順でランキングを表示する。  
→　幹事の負担が大幅に減り、みんなで楽しくお店選びができる  

## ユーザーの獲得について
- SNS（X / Instagram）でのシェア
- LINEでURL共有できるため、日常利用の流入が自然に発生

## サービスの差別化ポイント・推しポイント
- 「食べログ」や「Google Map」は「検索」はできるが 複数人で相談・比較する機能がない。
- LINEグループでは情報が流れて 比較しにくい。

## 機能候補
### MVPリリース
- 会員登録・ログイン機能
- イベントの作成・編集・削除
- イベントごとのユニークURL発行
- 招待（＝URL共有のみ）
- お店候補の登録（店名、URL、メモ 等）
- お店へのいいね機能
- お店一覧の並び替え、ランキング表示（登録順 / いいね順）
- イベントページの閲覧・投稿（ログインなしでも可）

### 本リリース（予定）
- コメント機能
- 写真投稿機能
- カテゴリーやタグ（カフェ / ランチ / 夜ごはん）機能で絞り込み
- Google Map 埋め込み
- 通知機能（いいね・提案が増えたとき）
